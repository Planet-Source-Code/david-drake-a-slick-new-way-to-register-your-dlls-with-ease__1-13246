Overview
--------
These REG files will update your registry so that
DLL and OCX files can be easily registered and un-
registered with Windows.


Files
-----
RegisterOCX_NT.reg - Use this file for Windows NT
RegisterOCX_95.reg - Use this file for Windows 95,98,Me


Instructions
------------
Double click on the appropriate Reg file from 
Windows Explorer. This will update the computer's
registry. You will get a prompt indicating that
the registry changes were successful.

That's it. Now go find a .DLL or .OCX file and right-
click on it. Notice there are two options in the menu, 
Register and Un-Register.
